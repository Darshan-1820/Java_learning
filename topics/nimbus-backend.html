Setup Checklist

  1. Install MySQL — Download MySQL Community Server. During install, set root password (remember it for db.properties).
  2. Run the SQL script — Open MySQL Workbench or terminal, paste the SQL setup script from Section 2 below. This creates the database, tables, and seed data.
  3. Create Maven project in Eclipse — File → New → Maven Project → Group ID: com.nimbus, Artifact ID: nimbus-backend. Replace the generated pom.xml with the one from Section 3.
  4. Create packages — Right-click src/main/java → New → Package. Create: com.nimbus.model, com.nimbus.util, com.nimbus.dao, com.nimbus.exception, com.nimbus.service, com.nimbus.app.
  5. Create db.properties — Right-click src/main/resources → New → File → name it db.properties. Paste the config from Section 4.
  6. Copy all 15 Java files — For each file below, create the class in the correct package and paste the code.
  7. Update Maven — Right-click project → Maven → Update Project (Alt+F5). This downloads the MySQL connector JAR.
  8. Run — Right-click NimbusApp.java → Run As → Java Application.

-- ============================
-- NIMBUS DATABASE SETUP
-- ============================

CREATE DATABASE IF NOT EXISTS nimbus_db;
USE nimbus_db;

-- Users table
CREATE TABLE users (
    userId      VARCHAR(20)   NOT NULL,
    firstName   VARCHAR(25)   NOT NULL,
    middleName  VARCHAR(25),
    lastName    VARCHAR(25)   NOT NULL,
    fullName    VARCHAR(50)   NOT NULL,
    email       VARCHAR(100)  NOT NULL,
    countryCode VARCHAR(5)    DEFAULT '+91',
    mobile      VARCHAR(10)   NOT NULL,
    street      VARCHAR(100),
    zipCode     VARCHAR(10),
    city        VARCHAR(50),
    state       VARCHAR(50),
    country     VARCHAR(50)   DEFAULT 'India',
    password    VARCHAR(30)   NOT NULL,
    role        ENUM('customer', 'officer', 'support') NOT NULL DEFAULT 'customer',
    preferences VARCHAR(100),
    createdAt   TIMESTAMP     DEFAULT CURRENT_TIMESTAMP,

    PRIMARY KEY (userId),
    UNIQUE KEY uk_email (email),
    INDEX idx_role (role),
    INDEX idx_city_state (city, state)
);

-- Bookings table
CREATE TABLE bookings (
    bookingId              VARCHAR(20)    NOT NULL,
    userId                 VARCHAR(20)    NOT NULL,
    recName                VARCHAR(50)    NOT NULL,
    recAddress             VARCHAR(150)   NOT NULL,
    recPin                 VARCHAR(6)     NOT NULL,
    recMobile              VARCHAR(15)    NOT NULL,
    parWeightGram          INT            NOT NULL,
    parContentsDescription VARCHAR(200),
    parDeliveryType        ENUM('Standard', 'Express', 'Overnight') NOT NULL DEFAULT 'Standard',
    parPackingPreference   VARCHAR(50),
    parPickupTime          DATETIME,
    parDropoffTime         DATETIME,
    parServiceCost         DECIMAL(10,2)  NOT NULL,
    parPaymentTime         DATETIME,
    parStatus              ENUM('Booked', 'Picked up', 'In Transit', 'Delivered', 'Returned')
                           NOT NULL DEFAULT 'Booked',
    paymentMethod          VARCHAR(20),
    createdAt              TIMESTAMP      DEFAULT CURRENT_TIMESTAMP,

    PRIMARY KEY (bookingId),
    CONSTRAINT fk_booking_user FOREIGN KEY (userId) REFERENCES users(userId)
        ON UPDATE CASCADE ON DELETE RESTRICT,
    INDEX idx_userId (userId),
    INDEX idx_status (parStatus),
    INDEX idx_createdAt (createdAt),
    INDEX idx_deliveryType (parDeliveryType)
);

Section 07 — com.nimbus.util

DBConnection.java

The database connection factory. Reads credentials from db.properties, loads the MySQL driver with Class.forName(), and returns a Connection object.

com/nimbus/util/DBConnection.java 
package com.nimbus.util;

import java.io.IOException;
import java.io.InputStream;
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;
import java.util.Properties;

public class DBConnection {

    private static final String URL;
    private static final String USER;
    private static final String PASSWORD;

    // Static block — runs once when the class is first loaded
    static {
        Properties props = new Properties();
        try (InputStream input = DBConnection.class
                .getClassLoader()
                .getResourceAsStream("db.properties")) {

            if (input == null) {
                throw new RuntimeException("db.properties not found in resources folder");
            }
            props.load(input);

            // Load the MySQL driver class into memory
            Class.forName(props.getProperty("db.driver"));

        } catch (IOException | ClassNotFoundException e) {
            throw new RuntimeException("Failed to initialize database config: " + e.getMessage(), e);
        }

        URL      = props.getProperty("db.url");
        USER     = props.getProperty("db.user");
        PASSWORD = props.getProperty("db.password");
    }

    /**
     * Returns a new connection to the nimbus_db database.
     * Every DAO method calls this, uses the connection, and closes it.
     */
    public static Connection getConnection() throws SQLException {
        return DriverManager.getConnection(URL, USER, PASSWORD);
    }

    // Private constructor — prevents instantiation (utility class)
    private DBConnection() {
    }
}
Section 08 — com.nimbus.dao

UserDAO.java

All database operations for users. Every method uses PreparedStatement with try-with-resources for automatic cleanup.

com/nimbus/dao/UserDAO.java 
package com.nimbus.dao;

import com.nimbus.model.User;
import com.nimbus.util.DBConnection;

import java.sql.*;
import java.util.ArrayList;
import java.util.List;

public class UserDAO {

    // ── INSERT ──
    public boolean addUser(User user) {
        String sql = "INSERT INTO users (userId, fullName, email, mobile, password, role, city) "
                   + "VALUES (?, ?, ?, ?, ?, ?, ?)";

        try (Connection conn = DBConnection.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {

            ps.setString(1, user.getUserId());
            ps.setString(2, user.getFullName());
            ps.setString(3, user.getEmail());
            ps.setString(4, user.getMobile());
            ps.setString(5, user.getPassword());
            ps.setString(6, user.getRole());
            ps.setString(7, user.getCity());

            return ps.executeUpdate() > 0;

        } catch (SQLException e) {
            System.err.println("Error adding user: " + e.getMessage());
            return false;
        }
    }

    // ── LOGIN VALIDATION ──
    public User validateLogin(String email, String password) {
        String sql = "SELECT userId, fullName, email, mobile, password, role, city "
                   + "FROM users WHERE email = ? AND password = ?";

        try (Connection conn = DBConnection.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {

            ps.setString(1, email);
            ps.setString(2, password);

            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    return extractUser(rs);
                }
            }
        } catch (SQLException e) {
            System.err.println("Error validating login: " + e.getMessage());
        }
        return null;
    }

    // ── GET BY ID ──
    public User getUserById(String userId) {
        String sql = "SELECT userId, fullName, email, mobile, password, role, city "
                   + "FROM users WHERE userId = ?";

        try (Connection conn = DBConnection.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {

            ps.setString(1, userId);

            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    return extractUser(rs);
                }
            }
        } catch (SQLException e) {
            System.err.println("Error fetching user: " + e.getMessage());
        }
        return null;
    }

    // ── EMAIL UNIQUENESS CHECK ──
    public boolean isEmailUnique(String email) {
        String sql = "SELECT COUNT(*) FROM users WHERE email = ?";

        try (Connection conn = DBConnection.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {

            ps.setString(1, email);

            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    return rs.getInt(1) == 0;  // true if no user has this email
                }
            }
        } catch (SQLException e) {
            System.err.println("Error checking email: " + e.getMessage());
        }
        return false;
    }

    // ── GET SUPPORT USERS ──
    public List<User> getSupportUsers() {
        String sql = "SELECT userId, fullName, email, mobile, password, role, city "
                   + "FROM users WHERE role = 'support'";
        List<User> supportUsers = new ArrayList<>();

        try (Connection conn = DBConnection.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql);
             ResultSet rs = ps.executeQuery()) {

            while (rs.next()) {
                supportUsers.add(extractUser(rs));
            }
        } catch (SQLException e) {
            System.err.println("Error fetching support users: " + e.getMessage());
        }
        return supportUsers;
    }

    // ── HELPER: Build User object from ResultSet row ──
    private User extractUser(ResultSet rs) throws SQLException {
        User user = new User();
        user.setUserId(rs.getString("userId"));
        user.setFullName(rs.getString("fullName"));
        user.setEmail(rs.getString("email"));
        user.setMobile(rs.getString("mobile"));
        user.setPassword(rs.getString("password"));
        user.setRole(rs.getString("role"));
        user.setCity(rs.getString("city"));
        return user;
    }
}
UserDAO — Section-by-Section Breakdown
1
addUser(User user)
Takes a User object and INSERTs it into the users table. Uses PreparedStatement with ? placeholders to prevent SQL injection. The try-with-resources block auto-closes both the Connection and PreparedStatement even if an exception occurs. Returns true if one row was inserted (executeUpdate() > 0).
2
validateLogin(String email, String password)
Checks if a user exists with the given email AND password combination. If rs.next() returns true, a matching row exists — we build a User object from it. If no match, returns null. The nested try (ResultSet rs ...) ensures the ResultSet is also auto-closed.
3
getUserById(String userId)
Fetches a single user by their primary key. Same pattern as validateLogin: prepare, execute, check rs.next(), extract or return null. Used by NimbusApp for "View Profile" feature.
4
isEmailUnique(String email)
Uses SELECT COUNT(*) to check if the email already exists. Returns true only when count is 0 (email is unique). Called by UserService.registerUser() before inserting a new user.
5
getSupportUsers()
Returns a List<User> of all support agents. Notice the triple try-with-resources: Connection, PreparedStatement, AND ResultSet are all auto-closed. Uses while (rs.next()) because there could be multiple support users.
6
extractUser(ResultSet rs)

            ps.setString(3,  b.getRecName());

        try (Connection conn = DBConnection.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql)) {

            ps.setTimestamp(1, pickup != null ? Timestamp.valueOf(pickup) : null);
            ps.setTimestamp(2, dropoff != null ? Timestamp.valueOf(dropoff) : null);
            ps.setString(3, bookingId);

        List<Booking> bookings = new ArrayList<>();

        try (Connection conn = DBConnection.getConnection();
             PreparedStatement ps = conn.prepareStatement(sql);
             ResultSet rs = ps.executeQuery()) {

            while (rs.next()) {
                bookings.add(extractBooking(rs));
            }
        } catch (SQLException e) {
            System.err.println("Error fetching all bookings: " + e.getMessage());
        }
        return bookings;

    private Booking extractBooking(ResultSet rs) throws SQLException {
        Booking b = new Booking();
        b.setBookingId(rs.getString("bookingId"));
        b.setUserId(rs.getString("userId"));
        b.setRecName(rs.getString("recName"));
        b.setRecAddress(rs.getString("recAddress"));
        b.setRecPin(rs.getString("recPin"));

        b.setParWeightGram(rs.getInt("parWeightGram"));
        b.setParContentsDescription(rs.getString("parContentsDescription"));
        b.setParDeliveryType(rs.getString("parDeliveryType"));
        b.setParPackingPreference(rs.getString("parPackingPreference"));

        Timestamp pickupTs = rs.getTimestamp("parPickupTime");
        b.setParPickupTime(pickupTs != null ? pickupTs.toLocalDateTime() : null);

        Timestamp dropTs = rs.getTimestamp("parDropoffTime");
        b.setParDropoffTime(dropTs != null ? dropTs.toLocalDateTime() : null);

        b.setParServiceCost(rs.getDouble("parServiceCost"));
        b.setParStatus(rs.getString("parStatus"));
        b.setPaymentMethod(rs.getString("paymentMethod"));
        return b;
    }

package com.nimbus.service;

import com.nimbus.dao.BookingDAO;
import com.nimbus.model.Booking;

/**
 * Processes a booking insertion in a background thread.
 * Demonstrates Java threading — the UI doesn't freeze
 * while the database insert happens.
 */
public class BookingThread extends Thread {

    private final Booking booking;
    private final BookingDAO bookingDAO;

    public BookingThread(Booking booking) {
        this.booking    = booking;
        this.bookingDAO = new BookingDAO();
        // Name the thread for identification in logs
        setName("BookingThread-" + booking.getBookingId());
    }

    @Override
    public void run() {
        System.out.println("[" + getName() + "] Processing booking...");

        try {
            // Simulate network/processing delay (1 second)
            Thread.sleep(1000);
        } catch (InterruptedException e) {
            System.err.println("[" + getName() + "] Thread interrupted: " + e.getMessage());
            Thread.currentThread().interrupt();
            return;
        }

        // Insert booking into database
        boolean success = bookingDAO.addBooking(booking);

        if (success) {
            System.out.println("[" + getName() + "] Booking " + booking.getBookingId()
                             + " saved successfully!");
        } else {
            System.err.println("[" + getName() + "] Failed to save booking "
                             + booking.getBookingId());
        }
    }
}

/**
 * Base class for booking history.
 * Provides the formatted table display.
 * Subclasses implement the data-fetching logic.
 */
public abstract class BaseHistoryService {

    /**
     * Displays a list of bookings in a formatted table.
     */
    public void displayHistory(List<Booking> bookings) {
        if (bookings.isEmpty()) {
            System.out.println("  No bookings found.");
            return;
        }

        // Table header
        System.out.println();
        System.out.printf("  %-12s %-20s %-12s %-10s %-12s%n",
                "Booking ID", "Receiver", "Type", "Cost", "Status");
        System.out.println("  " + "-".repeat(70));

        // Table rows
        for (Booking b : bookings) {
            System.out.printf("  %-12s %-20s %-12s Rs.%-7.2f %-12s%n",
                    b.getBookingId(),
                    truncate(b.getRecName(), 18),
                    b.getParDeliveryType(),
                    b.getParServiceCost(),
                    b.getParStatus());
        }

        System.out.println("  " + "-".repeat(70));
        System.out.printf("  Total: %d booking(s)%n", bookings.size());
    }

    /**
     * Truncates a string if it exceeds maxLen, adding "..." at the end.
     */
    protected String truncate(String text, int maxLen) {
        if (text == null) return "";
        return text.length() > maxLen ? text.substring(0, maxLen - 3) + "..." : text;
    }
}

        System.out.print("  Choose (1-3): ");

            case 2 -> "Express";

            default -> "Standard";

        System.out.println("  Payment Method:");
        System.out.println("    1. UPI");
        System.out.println("    2. Card");
        System.out.println("    3. Net Banking");
        System.out.println("    4. Cash");
        System.out.print("  Choose (1-4): ");
        int payChoice = readInt();
        String paymentMethod = switch (payChoice) {
            case 1 -> "UPI";
            case 2 -> "Card";
            case 3 -> "Net Banking";
            case 4 -> "Cash";
            default -> "UPI";
        };

        // Calculate cost
        double cost = calculateCost(weight, deliveryType);
        String bookingId = generateBookingId();

        // Build the Booking object
        Booking booking = new Booking(
            bookingId, currentUser.getUserId(), recName, recAddress, recPin,
            recMobile, weight, contents, deliveryType, packing,
            null, null, cost, "Booked", paymentMethod
        );

        // Show summary
        System.out.println("\n  ── BOOKING SUMMARY ──");
        System.out.printf("  Booking ID   : %s%n", bookingId);
        System.out.printf("  Receiver     : %s%n", recName);

            }

        }
    }

    // HELPER: Calculate Cost
    // ════════════════════════════════════════════
    private static double calculateCost(int weightGrams, String deliveryType) {
        // Base cost: Rs. 50 + Rs. 5 per 100 grams
        double baseCost = 50.0 + (weightGrams / 100.0) * 5.0;

        // Delivery type multiplier
        double multiplier = switch (deliveryType) {
            case "Express"   -> 1.5;
            case "Overnight" -> 2.0;
            default          -> 1.0;  // Standard
        };

        return Math.round(baseCost * multiplier * 100.0) / 100.0;
    }

NimbusApp.java — Section-by-Section Breakdown
1
Static fields (Scanner, UserService, DAOs, currentUser)
All fields are static because every method in this class is static (called from main()). currentUser starts as null and gets set after a successful login. It acts as the "session" — every method can check who's logged in.
2
main() — Main menu loop
A while (running) loop shows the main menu repeatedly until the user chooses "Exit." Java 17's enhanced switch (case 1 -> instead of case 1:) makes the code cleaner. No break needed.
3
login() — Role-based routing
Calls userService.login() which internally calls UserDAO.validateLogin(). If successful, checks the user's role field and routes to the correct menu. Each role sees different options. This is RBAC (Role-Based Access Control).
4
register() — DuplicateEmailException handling
Wraps userService.registerUser() in try-catch. If the email exists, the DuplicateEmailException is caught and its message is shown to the user. This is a real use of custom checked exceptions — the compiler forced us to handle it.
5
bookParcel() — BookingThread usage
Collects all inputs, calculates cost, builds a Booking object, then creates a BookingThread and calls .start(). The thread.join() call makes the main thread wait for the booking thread to finish, so the user sees the success/failure message before returning to the menu.
6
trackParcel() — Polymorphism in action
Declares TrackingService tracker (the parent type). Assigns either CustomerTrackingService or OfficerTrackingService based on the role. Then calls tracker.trackParcel() and tracker.displayResult(). Java's dynamic dispatch ensures the correct overridden version runs. One method, different behaviors. That is polymorphism.
7
bookingHistory() — Inheritance in action
CustomerHistoryService extends BaseHistoryService. It calls getMyBookings() (defined in the subclass) which throws BookingNotFoundException if empty. Then calls displayHistory() (defined in the parent class). The subclass adds data-fetching, the parent provides the display. That is inheritance.
8
calculateCost() — Pricing logic
Base cost is Rs. 50 + Rs. 5 per 100g. Express multiplies by 1.5x, Overnight by 2x. Math.round(... * 100.0) / 100.0 rounds to 2 decimal places (standard trick for currency in Java without BigDecimal).
9
readInt() — Safe integer input
Reads a line as String, then parses to int. If the user types "abc" instead of a number, NumberFormatException is caught and -1 is returned. This prevents the entire application from crashing on bad input. Never use scanner.nextInt() directly — it leaves a newline in the buffer and causes issues.
