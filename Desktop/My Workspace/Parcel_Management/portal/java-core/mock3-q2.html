Area-Perimeter of a Rectangle

Minor 10–15 min

Question Statement

Create a class Solution with the name AreaCalculation. This method will return area and perimeter based on the input given as Length and Breadth [the values are in meters] of a rectangle.

Formulas:

  • Area = Length * Breadth
  • Perimeter = 2 * (Length + Breadth)
Analysis

What This Question Tests

This is the simplest PRA question you will ever see. Pure math, no OOP, no collections. Just read two numbers and print two results.

What You Need
  • Scanner — read two integers
  • Basic math — multiplication and addition
  • Output format — exact format with "sq.m" and "m" units

No classes to create. No getters/setters. No ArrayList. No filtering. Just math and formatting. This should take under 5 minutes.

Logic

The Two Formulas

Area

Length × Breadth

Example: 5 × 10 = 50 sq.m

Perimeter

2 × (Length + Breadth)

Example: 2 × (5 + 10) = 30 m

Just two lines of logic 
int area = length * breadth;
int perimeter = 2 * (length + breadth);
Full Code

Complete Solution

Copy-paste ready. The entire program is under 15 lines.

Solution.java — Complete 
import java.util.Scanner;

public class Solution {
    public static void AreaCalculation(int length, int breadth) {
        int area = length * breadth;
        int perimeter = 2 * (length + breadth);
        System.out.println(area + " sq.m");
        System.out.println(perimeter + " m");
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int length = sc.nextInt();
        int breadth = sc.nextInt();
        AreaCalculation(length, breadth);
    }
}
Line-by-line explanation
1
public static void AreaCalculation(int length, int breadth)
A static method named exactly as the question asks. Takes two int parameters. Returns void because it prints directly.
2
int area = length * breadth;
Area formula. Both values are int, so the result is int. No decimals needed.
3
int perimeter = 2 * (length + breadth);
Perimeter formula. Parentheses ensure addition happens before multiplication.
4
System.out.println(area + " sq.m");
Print area followed by " sq.m". Note the space before "sq.m" — this must match exactly.
5
System.out.println(perimeter + " m");
Print perimeter followed by " m". Again, space before "m" is required.

Watch Out

Common Mistakes

Wrong

double area = length * breadth;

Output: 50.0 sq.m

Expected output is "50 sq.m" (no decimal). Using double adds ".0" and fails the test case.

Correct

int area = length * breadth;

Output: 50 sq.m

int gives clean integer output without decimal points.

Mistake What Goes Wrong Fix
Using double for area/perimeter Output shows "50.0 sq.m" instead of "50 sq.m" Use int — input values are integers, results will be integers
Missing space before unit Output shows "50sq.m" instead of "50 sq.m" Make sure string is " sq.m" with leading space
Wrong perimeter formula Forgetting the 2 * — output shows 15 instead of 30 Perimeter = 2 * (length + breadth), not just length + breadth
Output Format Is Everything

HackerRank compares your output character by character. One extra space, one missing unit, one ".0" where it shouldn't be — and the test case fails. Always match the expected output exactly.
