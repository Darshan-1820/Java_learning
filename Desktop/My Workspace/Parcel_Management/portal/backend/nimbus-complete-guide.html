01 — Overview

Project Overview

Nimbus is a Parcel Management System with 4 roles: Customer, Officer, Support, and Admin. Built across 4 sprints, progressing from console app to full REST APIs.

Roles

RoleWhat They Can Do
CustomerRegister, login, book parcels, track parcels, raise issues
OfficerLogin, view all bookings, update delivery status, track parcels
SupportLogin, view all issues, assign/resolve/reopen issues
AdminLogin, manage all users, view all data, system oversight

Tech Stack

Spring Boot3.2.5Application framework (auto-config, embedded Tomcat)
H2 Database2.2.224In-memory database (no install needed)
JPA / Hibernatevia Spring DataORM — Java objects map to database tables

Create the project from scratch in Eclipse.

File → New → Maven Project

  • Check "Create a simple project (skip archetype selection)"
  • Group ID: com.nimbus
  • Artifact ID: nimbus-backend
  • Version: 1.0.0
  • Packaging: jar
  • Click Finish

Step 2: Replace pom.xml

Open pom.xml in the project root. Delete ALL content and paste the pom.xml from Section 05 below.

04 — Troubleshooting

Real Errors We Hit

These are actual errors from our Eclipse setup. If you hit any of these, the fix is right here.

ErrorFix
Blank line before <?xml in pom.xml <?xml must be on line 1, column 1. No blank lines, no spaces before it.
DB_CLOSE_ON_EXIT not supported H2 2.2.224 dropped this setting. Remove it from the datasource URL. Use: jdbc:h2:mem:nimbusdb;MODE=MySQL
com.nimbus package not visible Eclipse hides empty packages by default. Create any class inside the package to make it appear. Or: click the triangle dropdown in Package Explorer → Filters → uncheck "Empty packages".
JPA entities not found Add spring.jpa.defer-datasource-initialization=true to application.properties. This makes JPA create tables before SQL scripts run.
Table not found (H2) Use lowercase table names in schema.sql when using JPA. H2 is case-sensitive by default.
Port 8080 already in use Another app is using port 8080. Either kill it (Task Manager → find java.exe) or change server.port=8081 in application.properties.
JSP not rendering (shows raw code) The tomcat-embed-jasper dependency is missing from pom.xml. Add it and do Maven → Update Project.

schema.sql

Creates all 3 database tables on startup.

src/main/resources/schema.sql 
CREATE TABLE IF NOT EXISTS tbl_user_profile (
    user_id VARCHAR(20) PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    middle_name VARCHAR(50),
    last_name VARCHAR(50) NOT NULL,
    full_name VARCHAR(150) NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    country_code VARCHAR(5),
    mobile VARCHAR(15) NOT NULL,
    street VARCHAR(200),
    zip_code VARCHAR(10),
    city VARCHAR(50),
    state VARCHAR(50),
    country VARCHAR(50),
    password VARCHAR(100) NOT NULL,
    role VARCHAR(20) NOT NULL,
    preferences VARCHAR(500),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE IF NOT EXISTS tbl_booking (
    booking_id VARCHAR(30) PRIMARY KEY,
    user_id VARCHAR(20) NOT NULL,
    rec_name VARCHAR(100) NOT NULL,
    rec_address VARCHAR(300),
    rec_pin VARCHAR(10),
    rec_mobile VARCHAR(15),
    par_weight_gram INT,
    par_contents_description VARCHAR(300),
    par_delivery_type VARCHAR(20),
    par_packing_preference VARCHAR(30),
    par_status VARCHAR(30) DEFAULT 'Booked',
    par_service_cost DOUBLE,
    payment_method VARCHAR(30),
    par_pickup_time TIMESTAMP,
    par_dropoff_time TIMESTAMP,
    par_payment_time TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES tbl_user_profile(user_id)
);

CREATE TABLE IF NOT EXISTS tbl_issue (
    issue_id VARCHAR(30) PRIMARY KEY,
    user_id VARCHAR(20) NOT NULL,
    booking_id VARCHAR(30),
    subject VARCHAR(200) NOT NULL,
    description VARCHAR(1000),
    severity VARCHAR(10) DEFAULT 'Low',
    status VARCHAR(20) DEFAULT 'Open',
    assigned_to VARCHAR(20),
    notes VARCHAR(2000),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES tbl_user_profile(user_id)
);

import jakarta.servlet.http.HttpServletRequest;

import jakarta.servlet.http.HttpSession;
import org.springframework.stereotype.Component;
import java.io.IOException;
import java.util.Optional;

@Component
public class LoginServlet extends HttpServlet {

    private final UserService userService;

    public LoginServlet(UserService userService) {
        this.userService = userService;
    }

    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        req.getRequestDispatcher("/WEB-INF/jsp/login.jsp").forward(req, resp);
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp)
            throws ServletException, IOException {
        String userId = req.getParameter("userId");
        String password = req.getParameter("password");

        Optional<User> user = userService.login(userId, password);

        if (user.isPresent()) {
            HttpSession session = req.getSession();
            session.setAttribute("user", user.get());
            session.setAttribute("role", user.get().getRole());
            resp.sendRedirect("/servlet/dashboard");
        } else {
            req.setAttribute("error", "Invalid User ID or Password");
            req.getRequestDispatcher("/WEB-INF/jsp/login.jsp").forward(req, resp);
        }
    }
}

package com.nimbus.controller;

import com.nimbus.dto.ApiResponse;
import com.nimbus.dto.LoginRequest;
import com.nimbus.dto.LoginResponse;
import com.nimbus.dto.RegisterRequest;
import com.nimbus.model.User;
import com.nimbus.service.UserService;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;
import java.util.List;
import java.util.Optional;

@RestController
@RequestMapping("/api/auth")
public class AuthController {

    private final UserService userService;

    public AuthController(UserService userService) {
        this.userService = userService;
    }

    @PostMapping("/login")
    public ResponseEntity<?> login(@RequestBody LoginRequest request) {
        Optional<User> user = userService.login(request.getUserId(), request.getPassword());
        if (user.isPresent()) {
            User u = user.get();
            LoginResponse response = new LoginResponse(
                u.getUserId(), u.getFullName(), u.getRole(), u.getEmail()
            );
            return ResponseEntity.ok(new ApiResponse("success", "Login successful", response));
        }
        return ResponseEntity.status(401)
            .body(new ApiResponse("error", "Invalid User ID or Password", null));
    }

    @PostMapping("/register")
    public ResponseEntity<?> register(@RequestBody RegisterRequest request) {
        try {
            User user = userService.register(request);
            return ResponseEntity.ok(new ApiResponse("success", "Registration successful", user));
        } catch (RuntimeException e) {
            return ResponseEntity.badRequest()
                .body(new ApiResponse("error", e.getMessage(), null));
        }
    }

    @GetMapping("/users")
    public ResponseEntity<?> getAllUsers() {
        List<User> users = userService.getAllUsers();
        return ResponseEntity.ok(new ApiResponse("success", "Found " + users.size() + " users", users));
    }

    @GetMapping("/users/{userId}")
    public ResponseEntity<?> getUser(@PathVariable String userId) {
        Optional<User> user = userService.getUserById(userId);
        if (user.isPresent()) {
            return ResponseEntity.ok(new ApiResponse("success", "User found", user.get()));
        }
        return ResponseEntity.status(404)
            .body(new ApiResponse("error", "User not found", null));
    }

    @GetMapping("/users/role/{role}")
    public ResponseEntity<?> getUsersByRole(@PathVariable String role) {
        List<User> users = userService.getUsersByRole(role);
        return ResponseEntity.ok(new ApiResponse("success", "Found " + users.size() + " users", users));
    }
}

src/app/api.ts 
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({ providedIn: 'root' })
export class ApiService {

  private baseUrl = 'http://localhost:8080/api';

  constructor(private http: HttpClient) {}

  login(userId: string, password: string): Observable<any> {
    return this.http.post(`${this.baseUrl}/auth/login`, { userId, password });
  }

  getAllUsers(): Observable<any> {
    return this.http.get(`${this.baseUrl}/auth/users`);
  }

  getBookingHistory(userId?: string, role?: string): Observable<any> {
    let params = `?role=${role || 'officer'}`;
    if (userId) params += `&userId=${userId}`;
    return this.http.get(`${this.baseUrl}/bookings/history${params}`);
  }

  getAllIssues(): Observable<any> {
    return this.http.get(`${this.baseUrl}/issues`);
  }

  getIssuesByUser(userId: string): Observable<any> {
    return this.http.get(`${this.baseUrl}/issues/user/${userId}`);
  }
}

export class Dashboard implements OnInit {
  user: any = null;
  bookings: any[] = [];
  issues: any[] = [];
  stats = { total: 0, transit: 0, delivered: 0, pending: 0 };

  constructor(private api: ApiService, private router: Router) {}

  ngOnInit() {
    const stored = sessionStorage.getItem('nimbus_user');
    if (!stored) { this.router.navigate(['/login']); return; }
    this.user = JSON.parse(stored);
    this.loadData();
  }

  loadData() {
    const role = this.user.role;
    const userId = this.user.userId;
    this.api.getBookingHistory(role === 'customer' ? userId : undefined, role).subscribe({
      next: (res: any) => {
        if (res.success) {
          this.bookings = res.data || [];
          this.stats.total = this.bookings.length;
          this.stats.transit = this.bookings.filter((b: any) => b.parStatus === 'In Transit').length;
          this.stats.delivered = this.bookings.filter((b: any) => b.parStatus === 'Delivered').length;
          this.stats.pending = this.bookings.filter((b: any) => b.parStatus === 'Booked').length;
        }
      }
    });
    if (role === 'customer') {
      this.api.getIssuesByUser(userId).subscribe({ next: (res: any) => { if (res.success) this.issues = res.data || []; } });
    } else {
      this.api.getAllIssues().subscribe({ next: (res: any) => { if (res.success) this.issues = res.data || []; } });
    }
  }

  logout() { sessionStorage.removeItem('nimbus_user'); this.router.navigate(['/login']); }

  getBadgeClass(status: string): string {
    switch (status) {
      case 'Booked': return 'badge-booked';
      case 'In Transit': return 'badge-transit';
      case 'Delivered': return 'badge-delivered';
      default: return 'badge-booked';
    }
  }
}

Deploy Angular to Spring Boot

How to put the compiled Angular files into your Eclipse project so Spring Boot serves them.

You Cannot Build Angular at TCS

Angular requires Node.js + npm to build. TCS machines don't have this. Build on your laptop first, then carry the compiled files.

Step 1: Build (on your laptop)

Terminal commands
cd nimbus-angular
ng build --output-path=dist

// Output goes to: nimbus-angular/dist/browser/
// Files: index.html, main-XXXX.js, styles-XXXX.css

Step 2: Copy to Eclipse Project

Copy the contents of dist/browser/ into your Eclipse project at:

Eclipse project path
nimbus-backend/
  src/main/resources/static/angular/
    index.html          ← copy here
    main-XXXX.js        ← copy here
    styles-XXXX.css     ← copy here
    favicon.ico         ← copy here

Step 3: Access in Browser

After running Spring Boot, open: http://localhost:8080/angular/index.html

Login with demo credentials → Angular dashboard loads data from REST APIs.

Important: base href

The index.html has <base href="/angular/">. This tells Angular that all its files are under /angular/. If you change the folder name, update this too.

Common Frontend Issues & Fixes

Problems you'll run into when integrating frontend with backend, and exact fixes.

IssueCauseFix
CORS error in Angular
blocked by CORS policy		 Angular runs on port 4200, Spring Boot on 8080. Different origins. Already handled by CorsConfig.java. If you serve Angular from Spring Boot's static/angular/, no CORS issue at all (same origin).
JSP shows raw code instead of HTML Missing tomcat-embed-jasper dependency in pom.xml. Add this to pom.xml:
<dependency><groupId>org.apache.tomcat.embed</groupId><artifactId>tomcat-embed-jasper</artifactId></dependency>
CSS not loading in JSP
Page looks plain white		 CSS files not in static/css/ or wrong path in JSP. Put CSS in src/main/resources/static/css/. Reference as /css/style.css (not ../css/style.css).
JSTL tags not working
<c:if> shows as text		 Missing JSTL dependency or wrong taglib URI. Add both JSTL dependencies (api + impl) to pom.xml. Use uri="jakarta.tags.core" (not the old http:// URI).
Angular routes show 404 on refresh Angular uses client-side routing. Spring Boot doesn't know about /angular/dashboard. Always navigate via Angular's router (from login). If you refresh, go to /angular/index.html first. Or add a Spring Boot controller that forwards unknown /angular/** routes to /angular/index.html.
Angular shows "Server error" Spring Boot is not running, or running on a different port. Start Spring Boot first. Check api.ts — baseUrl must match your server (default: http://localhost:8080/api).
JSP form data not reaching Servlet Input name attribute missing or wrong. Every <input> must have a name="..." attribute matching what the Servlet reads with request.getParameter("...").
Session lost after redirect Using response.sendRedirect() to an external URL loses the session. Always redirect within the same server. Use request.getContextPath() prefix.
