Section 01

Project Overview

This is the complete Nimbus Parcel Management web application built with Spring MVC, Servlets, JSP, and MySQL. Every file you need to run it in Eclipse + Tomcat is here.

What We Are Building

A parcel management system where customers can register, log in, book parcels, track deliveries, and view history. Delivery officers update parcel status. The entire backend is Spring MVC with JSP views and MySQL for persistence.

Why provided scope?

Servlet API and JSP API are already inside Tomcat. If you bundle them in your WAR, you get class conflicts. provided tells Maven: "I need this to compile, but don't package it — the server already has it."

Key Points

DispatcherServlet maps to / — it catches ALL requests. It uses AnnotationConfigWebApplicationContext instead of XML, loading our Java-based config from WebConfig.java.

AuthFilter protects /dashboard/*, /booking/*, /tracking/*, /delivery/*, and /history/*. Login and register pages are open.

Section 05

WebConfig.java

Spring MVC Java-based configuration. Sets up component scanning, view resolver, and the MySQL DataSource.

com/nimbus/config/WebConfig.java 
package com.nimbus.config;

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.ComponentScan;
import org.springframework.context.annotation.Configuration;
import org.springframework.jdbc.core.JdbcTemplate;
import org.springframework.jdbc.datasource.DriverManagerDataSource;
import org.springframework.web.servlet.ViewResolver;
import org.springframework.web.servlet.config.annotation.EnableWebMvc;
import org.springframework.web.servlet.config.annotation.ResourceHandlerRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurer;
import org.springframework.web.servlet.view.InternalResourceViewResolver;
import org.springframework.web.servlet.view.JstlView;

import javax.sql.DataSource;

@Configuration
@EnableWebMvc
@ComponentScan(basePackages = "com.nimbus")
public class WebConfig implements WebMvcConfigurer {

    // View Resolver: maps view names to JSP files
    // return "login" --> /WEB-INF/views/login.jsp
    @Bean
    public ViewResolver viewResolver() {
        InternalResourceViewResolver resolver =
            new InternalResourceViewResolver();
        resolver.setViewClass(JstlView.class);
        resolver.setPrefix("/WEB-INF/views/");
        resolver.setSuffix(".jsp");
        return resolver;
    }

    // Serve static resources (CSS, JS, images)
    @Override
    public void addResourceHandlers(ResourceHandlerRegistry registry) {
        registry.addResourceHandler("/css/**")
                .addResourceLocations("/css/");
        registry.addResourceHandler("/js/**")
                .addResourceLocations("/js/");
    }

    // MySQL DataSource
    @Bean
    public DataSource dataSource() {
        DriverManagerDataSource ds = new DriverManagerDataSource();
        ds.setDriverClassName("com.mysql.cj.jdbc.Driver");
        ds.setUrl("jdbc:mysql://localhost:3306/nimbus_db");
        ds.setUsername("root");
        ds.setPassword("root");
        return ds;
    }

    // JdbcTemplate for DAOs
    @Bean
    public JdbcTemplate jdbcTemplate(DataSource dataSource) {
        return new JdbcTemplate(dataSource);
    }
}
Line-by-line explanation
1
@Configuration
Marks this class as a Spring config class (replaces XML config)
2
@EnableWebMvc
Activates Spring MVC — registers HandlerMapping, converters, validators
3
@ComponentScan("com.nimbus")
Tells Spring to scan this package for @Controller, @Service, @Repository, @Component
4
InternalResourceViewResolver
When controller returns "login", resolver builds path: /WEB-INF/views/ + login + .jsp
5
DriverManagerDataSource
Simple DataSource that creates a new MySQL connection each time (fine for learning; use connection pool in production)

DAO Classes

Data Access Objects use Spring's JdbcTemplate to execute SQL queries against MySQL.

UserDAO.java

com/nimbus/dao/UserDAO.java 
package com.nimbus.dao;

import com.nimbus.model.User;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.jdbc.core.JdbcTemplate;
import org.springframework.jdbc.core.RowMapper;
import org.springframework.stereotype.Repository;

import java.sql.ResultSet;
import java.sql.SQLException;
import java.util.List;

@Repository
public class UserDAO {

    @Autowired
    private JdbcTemplate jdbcTemplate;

    // RowMapper converts each database row into a User object
    private final RowMapper<User> userMapper = (rs, rowNum) -> {
        User u = new User();
        u.setId(rs.getInt("id"));
        u.setUsername(rs.getString("username"));
        u.setPassword(rs.getString("password"));
        u.setFullName(rs.getString("full_name"));
        u.setEmail(rs.getString("email"));
        u.setPhone(rs.getString("phone"));
        u.setRole(rs.getString("role"));
        return u;
    };

    public User findByUsername(String username) {
        String sql = "SELECT * FROM users WHERE username = ?";
        List<User> results = jdbcTemplate.query(sql, userMapper, username);
        return results.isEmpty() ? null : results.get(0);
    }

    public User authenticate(String username, String password) {
        String sql = "SELECT * FROM users WHERE username = ? AND password = ?";
        List<User> results = jdbcTemplate.query(sql, userMapper,
                                                  username, password);
        return results.isEmpty() ? null : results.get(0);
    }

    public void register(User user) {
        String sql = "INSERT INTO users (username, password, full_name, " +
                     "email, phone, role) VALUES (?, ?, ?, ?, ?, ?)";
        jdbcTemplate.update(sql, user.getUsername(), user.getPassword(),
                           user.getFullName(), user.getEmail(),
                           user.getPhone(), user.getRole());
    }

    public boolean usernameExists(String username) {
        String sql = "SELECT COUNT(*) FROM users WHERE username = ?";
        Integer count = jdbcTemplate.queryForObject(sql, Integer.class,
                                                     username);
        return count != null && count > 0;
    }
}

ParcelDAO.java

com/nimbus/dao/ParcelDAO.java 
package com.nimbus.dao;

import com.nimbus.model.Parcel;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.jdbc.core.JdbcTemplate;
import org.springframework.jdbc.core.RowMapper;
import org.springframework.stereotype.Repository;

import java.time.LocalDate;
import java.util.List;
import java.util.UUID;

@Repository
public class ParcelDAO {

    @Autowired
    private JdbcTemplate jdbcTemplate;

    private final RowMapper<Parcel> parcelMapper = (rs, rowNum) -> {
        Parcel p = new Parcel();
        p.setId(rs.getInt("id"));
        p.setTrackingId(rs.getString("tracking_id"));
        p.setSenderId(rs.getInt("sender_id"));
        p.setSenderName(rs.getString("sender_name"));
        p.setRecipientName(rs.getString("recipient_name"));
        p.setRecipientPhone(rs.getString("recipient_phone"));
        p.setRecipientAddress(rs.getString("recipient_address"));
        p.setRecipientCity(rs.getString("recipient_city"));
        p.setRecipientPincode(rs.getString("recipient_pincode"));
        p.setWeight(rs.getDouble("weight"));
        p.setParcelType(rs.getString("parcel_type"));
        p.setPrice(rs.getDouble("price"));
        p.setStatus(rs.getString("status"));
        p.setBookingDate(rs.getDate("booking_date").toLocalDate());
        if (rs.getDate("delivery_date") != null) {
            p.setDeliveryDate(rs.getDate("delivery_date").toLocalDate());
        }
        return p;
    };

    public String bookParcel(Parcel parcel) {
        String trackingId = "NIM" + UUID.randomUUID().toString()
                                       .substring(0, 8).toUpperCase();
        parcel.setTrackingId(trackingId);
        parcel.setStatus("BOOKED");
        parcel.setBookingDate(LocalDate.now());

        String sql = "INSERT INTO parcels (tracking_id, sender_id, " +
            "sender_name, recipient_name, recipient_phone, " +
            "recipient_address, recipient_city, recipient_pincode, " +
            "weight, parcel_type, price, status, booking_date) " +
            "VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)";

        jdbcTemplate.update(sql, parcel.getTrackingId(),
            parcel.getSenderId(), parcel.getSenderName(),
            parcel.getRecipientName(), parcel.getRecipientPhone(),
            parcel.getRecipientAddress(), parcel.getRecipientCity(),
            parcel.getRecipientPincode(), parcel.getWeight(),
            parcel.getParcelType(), parcel.getPrice(),
            parcel.getStatus(), parcel.getBookingDate());

        return trackingId;
    }

    public Parcel findByTrackingId(String trackingId) {
        String sql = "SELECT * FROM parcels WHERE tracking_id = ?";
        List<Parcel> results = jdbcTemplate.query(sql, parcelMapper,
                                                    trackingId);
        return results.isEmpty() ? null : results.get(0);
    }

    public List<Parcel> findBySenderId(int senderId) {
        String sql = "SELECT * FROM parcels WHERE sender_id = ? " +
                     "ORDER BY booking_date DESC";
        return jdbcTemplate.query(sql, parcelMapper, senderId);
    }

    public List<Parcel> findPendingDeliveries() {
        String sql = "SELECT * FROM parcels WHERE status != 'DELIVERED' " +
                     "ORDER BY booking_date ASC";
        return jdbcTemplate.query(sql, parcelMapper);
    }

    public void updateStatus(String trackingId, String newStatus) {
        String sql = "UPDATE parcels SET status = ? WHERE tracking_id = ?";
        jdbcTemplate.update(sql, newStatus, trackingId);
    }

    public void markDelivered(String trackingId) {
        String sql = "UPDATE parcels SET status = 'DELIVERED', " +
                     "delivery_date = ? WHERE tracking_id = ?";
        jdbcTemplate.update(sql, LocalDate.now(), trackingId);
    }

    public double calculatePrice(double weight, String type) {
        double base;
        switch (type) {
            case "DOCUMENT": base = 50;  break;
            case "SMALL":    base = 100; break;
            case "MEDIUM":   base = 200; break;
            case "LARGE":    base = 350; break;
            default:         base = 100;
        }
        return base + (weight * 15);
    }
}

LoginController.java

com/nimbus/controller/LoginController.java 
package com.nimbus.controller;

import com.nimbus.dao.UserDAO;
import com.nimbus.model.User;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestParam;

import javax.servlet.http.HttpSession;

@Controller
public class LoginController {

    @Autowired
    private UserDAO userDAO;

    @GetMapping("/login")
    public String showLoginForm() {
        return "login";
    }

    @PostMapping("/login")
    public String processLogin(
            @RequestParam String username,
            @RequestParam String password,
            HttpSession session,
            Model model) {

        User user = userDAO.authenticate(username, password);

        if (user == null) {
            model.addAttribute("error", "Invalid username or password");
            return "login";
        }

        // Store user in session
        session.setAttribute("loggedInUser", user);

        // Redirect based on role
        if ("OFFICER".equals(user.getRole())) {
            return "redirect:/delivery/pending";
        }
        return "redirect:/dashboard";
    }

    @GetMapping("/logout")
    public String logout(HttpSession session) {
        session.invalidate();
        return "redirect:/login?logout=true";
    }
}

RegisterController.java

com/nimbus/controller/RegisterController.java 
package com.nimbus.controller;

import com.nimbus.dao.UserDAO;
import com.nimbus.model.User;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.ModelAttribute;
import org.springframework.web.bind.annotation.PostMapping;

@Controller
public class RegisterController {

    @Autowired
    private UserDAO userDAO;

    @GetMapping("/register")
    public String showRegisterForm(Model model) {
        model.addAttribute("user", new User());
        return "register";
    }

    @PostMapping("/register")
    public String processRegistration(
            @ModelAttribute User user,
            Model model) {

        // Check if username already taken
        if (userDAO.usernameExists(user.getUsername())) {
            model.addAttribute("error", "Username already exists");
            model.addAttribute("user", user);
            return "register";
        }

        // Default role = CUSTOMER
        user.setRole("CUSTOMER");
        userDAO.register(user);

        return "redirect:/login?registered=true";
    }
}

BookingController.java

com/nimbus/controller/BookingController.java 
package com.nimbus.controller;

import com.nimbus.dao.ParcelDAO;
import com.nimbus.model.Parcel;
import com.nimbus.model.User;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.ModelAttribute;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestMapping;

import javax.servlet.http.HttpSession;

@Controller
@RequestMapping("/booking")
public class BookingController {

    @Autowired
    private ParcelDAO parcelDAO;

    @GetMapping("/new")
    public String showBookingForm(Model model) {
        model.addAttribute("parcel", new Parcel());
        return "booking";
    }

    @PostMapping("/new")
    public String processBooking(
            @ModelAttribute Parcel parcel,
            HttpSession session,
            Model model) {

        User user = (User) session.getAttribute("loggedInUser");
        parcel.setSenderId(user.getId());
        parcel.setSenderName(user.getFullName());

        // Calculate price based on weight and type
        double price = parcelDAO.calculatePrice(
            parcel.getWeight(), parcel.getParcelType());
        parcel.setPrice(price);

        // Book and get tracking ID
        String trackingId = parcelDAO.bookParcel(parcel);

        // Show invoice
        Parcel booked = parcelDAO.findByTrackingId(trackingId);
        model.addAttribute("parcel", booked);
        return "invoice";
    }
}

package com.nimbus.controller;

import com.nimbus.dao.ParcelDAO;
import com.nimbus.model.Parcel;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestParam;

@Controller
@RequestMapping("/tracking")
public class TrackingController {

    @Autowired
    private ParcelDAO parcelDAO;

    @GetMapping
    public String showTrackingForm() {
        return "tracking";
    }

    @PostMapping
    public String trackParcel(
            @RequestParam String trackingId,
            Model model) {

        Parcel parcel = parcelDAO.findByTrackingId(
            trackingId.trim().toUpperCase());

        if (parcel == null) {
            model.addAttribute("error",

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestParam;

import java.util.List;

@Controller
@RequestMapping("/delivery")
public class DeliveryController {

    @Autowired
    private ParcelDAO parcelDAO;

    @GetMapping("/pending")
    public String showPendingDeliveries(Model model) {
        List<Parcel> pending = parcelDAO.findPendingDeliveries();
        model.addAttribute("parcels", pending);
        return "delivery-pending";
    }

    @PostMapping("/update-status")
    public String updateStatus(
            @RequestParam String trackingId,
            @RequestParam String newStatus,
            Model model) {

        if ("DELIVERED".equals(newStatus)) {
            parcelDAO.markDelivered(trackingId);
        } else {
            parcelDAO.updateStatus(trackingId, newStatus);
        }

        model.addAttribute("success",
            "Parcel " + trackingId + " updated to " + newStatus);

        // Reload pending list
        List<Parcel> pending = parcelDAO.findPendingDeliveries();
        model.addAttribute("parcels", pending);
        return "delivery-pending";
    }
}

package com.nimbus.controller;

import com.nimbus.dao.ParcelDAO;

import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;

import javax.servlet.http.HttpSession;
import java.util.List;

@Controller
public class HistoryController {

    @Autowired
    private ParcelDAO parcelDAO;

    @GetMapping("/dashboard")
    public String showDashboard(HttpSession session, Model model) {
        User user = (User) session.getAttribute("loggedInUser");
        List<Parcel> parcels = parcelDAO.findBySenderId(user.getId());

        long total = parcels.size();
        long delivered = parcels.stream()
            .filter(p -> "DELIVERED".equals(p.getStatus())).count();
        long inTransit = total - delivered;

        model.addAttribute("user", user);
        model.addAttribute("totalParcels", total);
        model.addAttribute("delivered", delivered);
        model.addAttribute("inTransit", inTransit);
        model.addAttribute("recentParcels",
            parcels.size() > 5 ? parcels.subList(0, 5) : parcels);

        return "customer-dashboard";
    }

    @GetMapping("/history")
    public String showHistory(HttpSession session, Model model) {
        User user = (User) session.getAttribute("loggedInUser");
        List<Parcel> parcels = parcelDAO.findBySenderId(user.getId());
        model.addAttribute("parcels", parcels);
        return "history";
    }
}

AuthFilter

A Servlet Filter that protects all secured URLs. If no user is in the session, the request is redirected to the login page.

com/nimbus/filter/AuthFilter.java 
package com.nimbus.filter;

import javax.servlet.*;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import javax.servlet.http.HttpSession;
import java.io.IOException;

public class AuthFilter implements Filter {

    @Override
    public void init(FilterConfig filterConfig) {}

    @Override
    public void doFilter(ServletRequest request,
                         ServletResponse response,
                         FilterChain chain)
            throws IOException, ServletException {

        HttpServletRequest req = (HttpServletRequest) request;
        HttpServletResponse res = (HttpServletResponse) response;
        HttpSession session = req.getSession(false);

        boolean loggedIn = (session != null
            && session.getAttribute("loggedInUser") != null);

        if (loggedIn) {
            chain.doFilter(request, response);
        } else {
            res.sendRedirect(req.getContextPath() + "/login");
        }
    }

    @Override
    public void destroy() {}
}
How AuthFilter Works

web.xml maps this filter to /dashboard/*, /booking/*, etc. Every request to those URLs passes through doFilter() first. If loggedInUser is not in the session, the user is redirected to /login. Otherwise, chain.doFilter() lets the request continue to the controller.

Deploy on Tomcat in Eclipse

Step-by-step guide to run the Nimbus web app locally.

Prerequisites

  • JDK 11+ installed
  • Eclipse IDE for Enterprise Java Developers
  • Apache Tomcat 9 downloaded and extracted
  • MySQL 8 running with the database created (Section 11)

Steps

  1. Import project: File → Import → Existing Maven Projects → Browse to nimbus-web folder → Finish. Eclipse resolves all Maven dependencies automatically.
  2. Add Tomcat server: Window → Preferences → Server → Runtime Environments → Add → Apache Tomcat v9.0 → Browse to your Tomcat folder → Finish.
  3. Configure server: In the Servers view (bottom panel), right-click → New → Server → Tomcat v9.0 → Add nimbus-web to Configured → Finish.
  4. Update database credentials: Open WebConfig.java and set your MySQL username/password in the dataSource() method.
  5. Run: Right-click server → Start. Open browser → http://localhost:8080/nimbus-web/login
  6. Login: Use customer1 / pass123 for customer view, or officer1 / pass123 for the delivery officer view.
Common Issues

404 error: Make sure the context path matches. Right-click project → Properties → Web Project Settings → Context root should be nimbus-web.

ClassNotFoundException for MySQL driver: Run mvn clean install to download dependencies, or right-click project → Maven → Update Project.

Cannot connect to database: Verify MySQL is running on port 3306 and the credentials in WebConfig.java match your MySQL setup.
