Every project file across all phases. Copy into Eclipse and run. Organized by phase — find what you need, grab it, go.

Complete Eclipse Setup — Step by Step

Follow every step in order

Step 1: Create Maven Project in Eclipse

1. Open Eclipse → File → New → Maven Project
2. Check "Create a simple project (skip archetype)" → Next
3. Group ID: com.nimbus  |  Artifact ID: NimbusParcelSystem  |  Version: 1.0
4. Click Finish. Maven creates the folder structure automatically.

Step 2: Create Packages

Right-click src/main/java → New → Package. Create each of these:

• com.nimbus.model — Data classes (User, Booking)
• com.nimbus.dao — Database access (UserDAO, BookingDAO)
• com.nimbus.service — Business logic (UserService, TrackingService...)
• com.nimbus.exception — Custom exceptions
• com.nimbus.util — Helpers (DBConnection)
• com.nimbus.app — Main application class

Step 3: Add MySQL Dependency

Open pom.xml in your project root. Replace its content with the pom.xml below. Save. Maven auto-downloads the MySQL driver.

Step 4: Set Up MySQL Database

Open MySQL command line or MySQL Workbench. Copy and run the SQL setup script below. This creates the database, tables, and seed data.

Step 5: Create Java Files

For each file below: Right-click the correct package → New → Class → name it → paste the code. Start with models, then util, then exceptions, then DAOs, then services, then the main app.

Step 6: Update DBConnection

CRITICAL: Open DBConnection.java and change USER and PASSWORD to match YOUR MySQL credentials. Without this, nothing connects.

Step 7: Run

1. Right-click NimbusApp.java → Run As → Java Application
2. Console opens with the Nimbus menu. Login with: 260401-AK01 / Amit@123
3. If you get ClassNotFoundException → Maven dependency not downloaded. Right-click project → Maven → Update Project.
4. If you get Connection refused → MySQL not running or wrong credentials in DBConnection.java.

What each layer does:
model/ = What the data looks like (Java classes with fields matching DB columns)
dao/ = How to read/write from database (SQL queries via JDBC)
service/ = Business rules (email must be unique, customers can only see own bookings)
exception/ = Custom error types for specific failures
util/ = Shared tools (database connection)
app/ = The console menu that ties everything together

Database Setup

SQL

Run this in MySQL to create the database, tables, and seed data.

nimbus_setup.sql 
-- ============================================
-- NIMBUS DATABASE SETUP — Run this first!
-- ============================================

CREATE DATABASE IF NOT EXISTS nimbus_db;
USE nimbus_db;

-- Users table
CREATE TABLE tbl_UserProfile (
    User_ID         VARCHAR(20)   PRIMARY KEY,
    FullName        VARCHAR(50)   NOT NULL,
    Rec_Address     VARCHAR(200)  NOT NULL,
    Email           VARCHAR(100)  NOT NULL UNIQUE,
    MobileNumber    VARCHAR(15)   NOT NULL,
    Password        VARCHAR(30)   NOT NULL,
    Role            VARCHAR(10)   NOT NULL
        CHECK (Role IN ('customer', 'officer', 'support'))
);

-- Bookings table
CREATE TABLE Booking (
    Booking_ID                  VARCHAR(20)   PRIMARY KEY,
    User_ID                     VARCHAR(20)   NOT NULL,
    Rec_Name                    VARCHAR(50)   NOT NULL,
    Rec_Address                 VARCHAR(200)  NOT NULL,
    Rec_Pin                     VARCHAR(6)    NOT NULL,
    Rec_Mobile                  VARCHAR(15)   NOT NULL,
    Par_Weight_Gram             INT           NOT NULL,
    Par_Contents_Description    VARCHAR(200)  NOT NULL,
    Par_Delivery_Type           VARCHAR(20)   NOT NULL,
    Par_Packing_Preference      VARCHAR(20)   NOT NULL,
    Par_PickupTime              DATETIME,
    Par_DropoffTime             DATETIME,
    Par_ServiceCost             DECIMAL(10,2) NOT NULL,
    Par_PaymentTime             DATETIME,
    Par_Status                  VARCHAR(20)   NOT NULL DEFAULT 'Booked',
    Booking_Date                DATETIME      DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (User_ID) REFERENCES tbl_UserProfile(User_ID)
);

package com.nimbus.util;

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DBConnection {
    private static final String URL = "jdbc:mysql://localhost:3306/nimbus_db";
    private static final String USER = "root";
    private static final String PASSWORD = "password";  // CHANGE THIS

    public static Connection getConnection() throws SQLException {
        try {
            Class.forName("com.mysql.cj.jdbc.Driver");
        } catch (ClassNotFoundException e) {
            throw new SQLException("MySQL Driver not found! Add mysql-connector to Build Path.", e);
        }
        return DriverManager.getConnection(URL, USER, PASSWORD);
    }
}

UserDAO.java

com.nimbus.dao DAO

All database operations for users — register, login, find, email check. Uses PreparedStatement to prevent SQL injection.

com/nimbus/dao/UserDAO.java 
package com.nimbus.dao;

import com.nimbus.model.User;
import com.nimbus.util.DBConnection;
import java.sql.*;
import java.util.ArrayList;

public class UserDAO {

    public boolean addUser(User user) {
        String sql = "INSERT INTO tbl_UserProfile (User_ID, FullName, Rec_Address, Email, MobileNumber, Password, Role) VALUES (?, ?, ?, ?, ?, ?, ?)";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement pstmt = conn.prepareStatement(sql)) {
            pstmt.setString(1, user.getUserId());
            pstmt.setString(2, user.getFullName());
            pstmt.setString(3, user.getAddress());
            pstmt.setString(4, user.getEmail());
            pstmt.setString(5, user.getMobile());
            pstmt.setString(6, user.getPassword());
            pstmt.setString(7, user.getRole());
            return pstmt.executeUpdate() > 0;
        } catch (SQLException e) {
            System.err.println("Error adding user: " + e.getMessage());
            return false;
        }
    }

    public User validateLogin(String userId, String password) {
        String sql = "SELECT * FROM tbl_UserProfile WHERE User_ID = ? AND Password = ?";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement pstmt = conn.prepareStatement(sql)) {
            pstmt.setString(1, userId);
            pstmt.setString(2, password);
            ResultSet rs = pstmt.executeQuery();
            if (rs.next()) return extractUser(rs);
            return null;
        } catch (SQLException e) {
            System.err.println("Error validating login: " + e.getMessage());
            return null;
        }
    }

    public User getUserById(String userId) {
        String sql = "SELECT * FROM tbl_UserProfile WHERE User_ID = ?";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement pstmt = conn.prepareStatement(sql)) {
            pstmt.setString(1, userId);
            ResultSet rs = pstmt.executeQuery();
            if (rs.next()) return extractUser(rs);
            return null;
        } catch (SQLException e) {
            System.err.println("Error getting user: " + e.getMessage());
            return null;
        }
    }

    public boolean isEmailUnique(String email) {
        String sql = "SELECT COUNT(*) AS cnt FROM tbl_UserProfile WHERE Email = ?";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement pstmt = conn.prepareStatement(sql)) {
            pstmt.setString(1, email);
            ResultSet rs = pstmt.executeQuery();
            if (rs.next()) return rs.getInt("cnt") == 0;
            return true;
        } catch (SQLException e) { return false; }
    }

    public ArrayList<User> getSupportUsers() {
        String sql = "SELECT * FROM tbl_UserProfile WHERE Role = 'support'";
        ArrayList<User> users = new ArrayList<>();
        try (Connection conn = DBConnection.getConnection();
             Statement stmt = conn.createStatement();
             ResultSet rs = stmt.executeQuery(sql)) {
            while (rs.next()) users.add(extractUser(rs));
        } catch (SQLException e) { System.err.println("Error: " + e.getMessage()); }
        return users;
    }

    private User extractUser(ResultSet rs) throws SQLException {
        User u = new User();
        u.setUserId(rs.getString("User_ID"));
        u.setFullName(rs.getString("FullName"));
        u.setEmail(rs.getString("Email"));
        u.setMobile(rs.getString("MobileNumber"));
        u.setAddress(rs.getString("Rec_Address"));
        u.setPassword(rs.getString("Password"));
        u.setRole(rs.getString("Role"));
        return u;
    }
}

com/nimbus/dao/BookingDAO.java 
package com.nimbus.dao;

import com.nimbus.model.Booking;
import com.nimbus.util.DBConnection;
import java.sql.*;
import java.time.LocalDateTime;
import java.util.ArrayList;

public class BookingDAO {

    public boolean addBooking(Booking b) {
        String sql = "INSERT INTO Booking (Booking_ID, User_ID, Rec_Name, Rec_Address, Rec_Pin, Rec_Mobile, Par_Weight_Gram, Par_Contents_Description, Par_Delivery_Type, Par_Packing_Preference, Par_PickupTime, Par_DropoffTime, Par_ServiceCost, Par_PaymentTime, Par_Status) VALUES (?,?,?,?,?,?,?,?,?,?,?,?,?,?,?)";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement p = conn.prepareStatement(sql)) {
            p.setString(1, b.getBookingId()); p.setString(2, b.getUserId());
            p.setString(3, b.getRecName()); p.setString(4, b.getRecAddress());
            p.setString(5, b.getRecPin()); p.setString(6, b.getRecMobile());
            p.setInt(7, b.getParWeightGram()); p.setString(8, b.getParContentsDesc());
            p.setString(9, b.getParDeliveryType()); p.setString(10, b.getParPackingPref());
            p.setTimestamp(11, b.getParPickupTime() != null ? Timestamp.valueOf(b.getParPickupTime()) : null);
            p.setTimestamp(12, b.getParDropoffTime() != null ? Timestamp.valueOf(b.getParDropoffTime()) : null);
            p.setDouble(13, b.getParServiceCost());
            p.setTimestamp(14, Timestamp.valueOf(LocalDateTime.now()));
            p.setString(15, b.getParStatus());
            return p.executeUpdate() > 0;
        } catch (SQLException e) { System.err.println("Error adding booking: " + e.getMessage()); return false; }
    }

    public Booking getBookingById(String bookingId) {
        String sql = "SELECT * FROM Booking WHERE Booking_ID = ?";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement p = conn.prepareStatement(sql)) {
            p.setString(1, bookingId);
            ResultSet rs = p.executeQuery();
            if (rs.next()) return extractBooking(rs);
            return null;
        } catch (SQLException e) { System.err.println("Error: " + e.getMessage()); return null; }
    }

    public ArrayList<Booking> getBookingsByUser(String userId) {
        String sql = "SELECT * FROM Booking WHERE User_ID = ? ORDER BY Booking_Date DESC";
        ArrayList<Booking> list = new ArrayList<>();
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement p = conn.prepareStatement(sql)) {
            p.setString(1, userId);
            ResultSet rs = p.executeQuery();
            while (rs.next()) list.add(extractBooking(rs));
        } catch (SQLException e) { System.err.println("Error: " + e.getMessage()); }
        return list;
    }

    public ArrayList<Booking> getBookingsByDateRange(String userId, String start, String end) {
        String sql = "SELECT * FROM Booking WHERE User_ID = ? AND Booking_Date BETWEEN ? AND ? ORDER BY Booking_Date DESC";
        ArrayList<Booking> list = new ArrayList<>();
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement p = conn.prepareStatement(sql)) {
            p.setString(1, userId); p.setString(2, start); p.setString(3, end);
            ResultSet rs = p.executeQuery();
            while (rs.next()) list.add(extractBooking(rs));
        } catch (SQLException e) { System.err.println("Error: " + e.getMessage()); }
        return list;
    }

    public ArrayList<Booking> getAllBookings() {
        String sql = "SELECT * FROM Booking ORDER BY Booking_Date DESC";
        ArrayList<Booking> list = new ArrayList<>();
        try (Connection conn = DBConnection.getConnection();
             Statement s = conn.createStatement(); ResultSet rs = s.executeQuery(sql)) {
            while (rs.next()) list.add(extractBooking(rs));
        } catch (SQLException e) { System.err.println("Error: " + e.getMessage()); }
        return list;
    }

    public boolean updateStatus(String bookingId, String status) {
        String sql = "UPDATE Booking SET Par_Status = ? WHERE Booking_ID = ?";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement p = conn.prepareStatement(sql)) {
            p.setString(1, status); p.setString(2, bookingId);
            return p.executeUpdate() > 0;
        } catch (SQLException e) { System.err.println("Error: " + e.getMessage()); return false; }
    }

    public boolean updatePickupDrop(String bookingId, LocalDateTime pickup, LocalDateTime dropoff) {
        String sql = "UPDATE Booking SET Par_PickupTime = ?, Par_DropoffTime = ? WHERE Booking_ID = ?";
        try (Connection conn = DBConnection.getConnection();
             PreparedStatement p = conn.prepareStatement(sql)) {
            p.setTimestamp(1, Timestamp.valueOf(pickup));
            p.setTimestamp(2, Timestamp.valueOf(dropoff));
            p.setString(3, bookingId);
            return p.executeUpdate() > 0;
        } catch (SQLException e) { System.err.println("Error: " + e.getMessage()); return false; }
    }

    private Booking extractBooking(ResultSet rs) throws SQLException {
        Booking b = new Booking();
        b.setBookingId(rs.getString("Booking_ID")); b.setUserId(rs.getString("User_ID"));
        b.setRecName(rs.getString("Rec_Name")); b.setRecAddress(rs.getString("Rec_Address"));
        b.setRecPin(rs.getString("Rec_Pin")); b.setRecMobile(rs.getString("Rec_Mobile"));
        b.setParWeightGram(rs.getInt("Par_Weight_Gram"));
        b.setParContentsDesc(rs.getString("Par_Contents_Description"));
        b.setParDeliveryType(rs.getString("Par_Delivery_Type"));
        b.setParPackingPref(rs.getString("Par_Packing_Preference"));
        Timestamp pk = rs.getTimestamp("Par_PickupTime");
        if (pk != null) b.setParPickupTime(pk.toLocalDateTime());
        Timestamp dr = rs.getTimestamp("Par_DropoffTime");
        if (dr != null) b.setParDropoffTime(dr.toLocalDateTime());
        b.setParServiceCost(rs.getDouble("Par_ServiceCost"));
        Timestamp pay = rs.getTimestamp("Par_PaymentTime");
        if (pay != null) b.setParPaymentTime(pay.toLocalDateTime());
        b.setParStatus(rs.getString("Par_Status"));
        return b;
    }
}

package com.nimbus.app;

import com.nimbus.model.User;
import com.nimbus.model.Booking;

public class NimbusApp {
    private static Scanner scanner = new Scanner(System.in);
    private static UserDAO userDAO = new UserDAO();
    private static BookingDAO bookingDAO = new BookingDAO();
    private static User currentUser = null;

    public static void main(String[] args) {
        System.out.println("=== NIMBUS PARCEL SYSTEM ===");
        boolean running = true;
        while (running) {
            if (currentUser == null) running = showMainMenu();

    private static boolean showMainMenu() {
        System.out.println("\n1. Login  2. Register  3. Exit");
        System.out.print("Choose: ");
        int c = readInt();
        if (c == 1) login(); else if (c == 2) register(); else if (c == 3) return false;
        return true;
    }

    private static void login() {
        System.out.print("User ID: "); String id = scanner.nextLine().trim();
        System.out.print("Password: "); String pw = scanner.nextLine().trim();
        User u = userDAO.validateLogin(id, pw);
        if (u != null) { currentUser = u; System.out.println("Welcome, " + u.getFullName() + "!"); }
        else System.out.println("Invalid credentials.");
    }

    private static void register() {
        System.out.print("Name: "); String name = scanner.nextLine().trim();
        System.out.print("Email: "); String email = scanner.nextLine().trim();
        System.out.print("Mobile (10 digits): "); String mobile = scanner.nextLine().trim();
        System.out.print("Address: "); String addr = scanner.nextLine().trim();
        System.out.print("User ID (5-20 chars): "); String uid = scanner.nextLine().trim();
        System.out.print("Password: "); String pw = scanner.nextLine().trim();
        try {
            new UserService().registerUser(new User(uid, name, email, mobile, addr, pw, "customer"));
            System.out.println("Registration successful! User ID: " + uid);
        } catch (DuplicateEmailException e) { System.out.println("ERROR: " + e.getMessage()); }
    }

    private static void showCustomerMenu() {
        System.out.println("\n1. Book  2. Track  3. History  4. Support  5. Logout");
        System.out.print("Choose: ");
        int c = readInt();
        switch (c) {
            case 1: bookParcel(false); break; case 2: trackParcel(); break;
            case 3: bookingHistory(); break;
            case 4: System.out.println("Support: 1800-NIMBUS-00 | support@nimbus.com"); break;
            case 5: logout(); break;
        }
    }

    private static void showOfficerMenu() {
        System.out.println("\n1. Book  2. Track  3. Delivery Status  4. Pickup  5. History  6. Logout");
        System.out.print("Choose: ");
        int c = readInt();
        switch (c) {
            case 1: bookParcel(true); break; case 2: trackParcel(); break;
            case 3: updateDeliveryStatus(); break; case 4: schedulePickup(); break;
            case 5: officerHistory(); break; case 6: logout(); break;
        }
    }

    private static void showSupportMenu() {
        System.out.println("\n1. View Issues  2. Logout");
        System.out.print("Choose: "); int c = readInt();
        if (c == 2) logout();
    }

    private static void bookParcel(boolean isOfficer) {
        String senderId = isOfficer ? promptAndGet("Customer ID: ") : currentUser.getUserId();
        String recName = promptAndGet("Receiver Name: ");
        String recAddr = promptAndGet("Receiver Address: ");
        String recPin = promptAndGet("Receiver PIN: ");
        String recMob = promptAndGet("Receiver Mobile: ");
        System.out.print("Weight (grams): "); int weight = readInt();
        String contents = promptAndGet("Contents: ");
        System.out.print("Delivery (1=Standard 2=Express 3=Overnight): "); int dt = readInt();
        String delType = dt == 2 ? "Express" : dt == 3 ? "Overnight" : "Standard";
        System.out.print("Packing (1=Std 2=Custom 3=Eco 4=Fragile): "); int pk = readInt();
        String packing = pk == 2 ? "Custom" : pk == 3 ? "Eco-friendly" : pk == 4 ? "Fragile" : "Standard";

        double cost = calcCost(weight, delType, packing);
        System.out.printf("Cost: Rs. %.2f. Confirm? (y/n): ", cost);
        if (scanner.nextLine().trim().equalsIgnoreCase("y")) {
            String bid = "NMB-" + String.format("%02d%02d%02d", LocalDateTime.now().getYear() % 100,
                LocalDateTime.now().getMonthValue(), LocalDateTime.now().getDayOfMonth()) + "-" + ((int)(Math.random()*9000)+1000);
            Booking b = new Booking(); b.setBookingId(bid); b.setUserId(senderId);
            b.setRecName(recName); b.setRecAddress(recAddr); b.setRecPin(recPin); b.setRecMobile(recMob);
            b.setParWeightGram(weight); b.setParContentsDesc(contents);
            b.setParDeliveryType(delType); b.setParPackingPref(packing);
            b.setParServiceCost(cost); b.setParStatus("Booked");
            BookingThread thread = new BookingThread(b);
            thread.start();
            try { thread.join(); } catch (InterruptedException e) { }
            if (thread.isSuccess()) System.out.println("Booking confirmed: " + bid);
        }
    }

    private static void trackParcel() {
        String bid = promptAndGet("Booking ID: ");
        TrackingService tracker = currentUser.getRole().equals("customer")
            ? new CustomerTrackingService() : new OfficerTrackingService();
        tracker.displayResult(tracker.trackParcel(bid, currentUser.getUserId()));
    }

    private static void bookingHistory() {
        CustomerHistoryService hs = new CustomerHistoryService();
        try { hs.displayHistory(hs.getMyBookings(currentUser.getUserId())); }
        catch (BookingNotFoundException e) { System.out.println(e.getMessage()); }
    }

    private static void officerHistory() {
        String uid = promptAndGet("Customer ID (or Enter for all): ");
        ArrayList<Booking> list = uid.isEmpty() ? bookingDAO.getAllBookings() : bookingDAO.getBookingsByUser(uid);
        new BaseHistoryService().displayHistory(list);
    }

    private static void updateDeliveryStatus() {
        String bid = promptAndGet("Booking ID: ");
        Booking b = bookingDAO.getBookingById(bid);
        if (b == null) { System.out.println("Not found."); return; }
        System.out.println("Current: " + b.getParStatus());
        System.out.print("New (1=Booked 2=Picked Up 3=In Transit 4=Delivered 5=Returned): "); int c = readInt();
        String[] s = {"Booked","Picked Up","In Transit","Delivered","Returned"};
        if (c >= 1 && c <= 5 && bookingDAO.updateStatus(bid, s[c-1]))
            System.out.println("Updated to: " + s[c-1]);
    }

    private static void schedulePickup() {
        String bid = promptAndGet("Booking ID: ");
        Booking b = bookingDAO.getBookingById(bid);
        if (b == null) { System.out.println("Not found."); return; }
        DateTimeFormatter fmt = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm");
        try {
            LocalDateTime pk = LocalDateTime.parse(promptAndGet("Pickup (yyyy-MM-dd HH:mm): "), fmt);
            LocalDateTime dr = LocalDateTime.parse(promptAndGet("Dropoff (yyyy-MM-dd HH:mm): "), fmt);
            if (dr.isBefore(pk)) { System.out.println("Dropoff must be after pickup."); return; }
            if (bookingDAO.updatePickupDrop(bid, pk, dr)) System.out.println("Schedule updated.");
        } catch (Exception e) { System.out.println("Invalid format."); }
    }

    private static void logout() {
        System.out.println("Goodbye, " + currentUser.getFullName() + "!");
        currentUser = null;
    }

    private static double calcCost(int w, String d, String p) {
        double c = w * 0.15;
        if ("Express".equals(d)) c *= 1.8; else if ("Overnight".equals(d)) c *= 3.0;
        if ("Custom".equals(p)) c += 50; else if ("Eco-friendly".equals(p)) c += 30; else if ("Fragile".equals(p)) c += 80;
        return Math.max(c, 50.0);
    }

    private static String promptAndGet(String prompt) { System.out.print(prompt); return scanner.nextLine().trim(); }
    private static int readInt() { try { return Integer.parseInt(scanner.nextLine().trim()); } catch (Exception e) { return -1; } }
}

package com.nimbus.controller;

import com.nimbus.dao.UserDAO;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.*;
import org.springframework.web.bind.annotation.*;
import java.util.Map;

@RestController
@RequestMapping("/api")
@CrossOrigin(origins = "*")
public class UserController {

    @Autowired
    private UserService userService;

    @Autowired
    private UserDAO userDAO;

    @PostMapping("/register")
    public ResponseEntity<?> register(@RequestBody User user) {
        try {
            boolean success = userService.registerUser(user);
            if (success) {
                return ResponseEntity.status(HttpStatus.CREATED)
                    .body(Map.of("message", "Customer registered successfully",
                                 "userId", user.getUserId()));
            }
            return ResponseEntity.badRequest()
                .body(Map.of("error", "Registration failed"));
        } catch (DuplicateEmailException e) {
            return ResponseEntity.badRequest()
                .body(Map.of("error", e.getMessage()));
        }
    }

    @PostMapping("/login")
    public ResponseEntity<?> login(@RequestBody Map<String, String> credentials) {
        String userId = credentials.get("userId");
        String password = credentials.get("password");

        User user = userDAO.validateLogin(userId, password);
        if (user != null) {
            return ResponseEntity.ok(Map.of(
                "message", "Customer logged in successfully",
                "userId", user.getUserId(),
                "fullName", user.getFullName(),
                "role", user.getRole()
            ));
        }
        return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
            .body(Map.of("error", "Invalid credentials"));
    }
}

package com.nimbus.controller;

import com.nimbus.dao.BookingDAO;
import com.nimbus.model.Booking;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.*;
import org.springframework.web.bind.annotation.*;
import java.time.LocalDateTime;
import java.util.*;

@RestController
@RequestMapping("/api/bookings")
@CrossOrigin(origins = "*")
public class BookingController {

    @Autowired
    private BookingDAO bookingDAO;

    @PostMapping
    public ResponseEntity<?> createBooking(@RequestBody Booking booking) {
        booking.setParPaymentTime(LocalDateTime.now());
        booking.setParStatus("Booked");
        boolean success = bookingDAO.addBooking(booking);

        if (success) {
            return ResponseEntity.status(HttpStatus.CREATED)
                .body(Map.of("message", "Booked a service successfully",
                             "bookingId", booking.getBookingId()));
        }
        return ResponseEntity.badRequest()
            .body(Map.of("error", "Booking failed"));
    }

    @GetMapping("/{bookingId}")
    public ResponseEntity<?> trackBooking(@PathVariable String bookingId) {
        Booking booking = bookingDAO.getBookingById(bookingId);
        if (booking != null) {
            return ResponseEntity.ok(booking);
        }
        return ResponseEntity.notFound().build();
    }

    @GetMapping
    public ResponseEntity<?> getBookings(
            @RequestParam(required = false) String userId,
            @RequestParam(required = false) String startDate,
            @RequestParam(required = false) String endDate) {

        ArrayList<Booking> bookings;

        if (userId != null && startDate != null && endDate != null) {
            bookings = bookingDAO.getBookingsByDateRange(userId, startDate, endDate);
        } else if (userId != null) {
            bookings = bookingDAO.getBookingsByUser(userId);
        } else {
            bookings = bookingDAO.getAllBookings();
        }

        return ResponseEntity.ok(bookings);
    }

    @PutMapping("/{bookingId}/status")
    public ResponseEntity<?> updateStatus(@PathVariable String bookingId,
                                            @RequestBody Map<String, String> body) {
        boolean updated = bookingDAO.updateStatus(bookingId, body.get("status"));
        if (updated) {
            return ResponseEntity.ok(Map.of("message", "Delivery Status updated successfully"));
        }
        return ResponseEntity.notFound().build();
    }

    @PutMapping("/{bookingId}/pickup")
    public ResponseEntity<?> updatePickup(@PathVariable String bookingId,
                                            @RequestBody Map<String, String> body) {
        LocalDateTime pickup = LocalDateTime.parse(body.get("pickupTime"));
        LocalDateTime dropoff = LocalDateTime.parse(body.get("dropoffTime"));
        boolean updated = bookingDAO.updatePickupDrop(bookingId, pickup, dropoff);
        if (updated) {
            return ResponseEntity.ok(Map.of("message", "Pick up and drop updated successfully"));
        }
        return ResponseEntity.notFound().build();
    }
}

import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({ providedIn: 'root' })
export class BookingService {
  private apiUrl = 'http://localhost:8080/api/bookings';

  constructor(private http: HttpClient) {}

  createBooking(booking: any): Observable<any> {
    return this.http.post(this.apiUrl, booking);
  }

  trackBooking(bookingId: string): Observable<any> {
    return this.http.get(`${this.apiUrl}/${bookingId}`);
  }

  getBookingsByUser(userId: string): Observable<any> {
    return this.http.get(`${this.apiUrl}?userId=${userId}`);
  }

  getAllBookings(): Observable<any> {
    return this.http.get(this.apiUrl);
  }

  updateStatus(bookingId: string, status: string): Observable<any> {
    return this.http.put(`${this.apiUrl}/${bookingId}/status`, { status });
  }

  updatePickup(bookingId: string, pickupTime: string, dropoffTime: string): Observable<any> {
    return this.http.put(`${this.apiUrl}/${bookingId}/pickup`, { pickupTime, dropoffTime });
  }
}
