Mock 4 — Question 2

Consecutive Vowels

Minor 10–15 min

Question Statement

Write a Java program to find the count of consecutive vowels in a given string.

A "consecutive vowel" group is where 2 or more vowels appear next to each other (e.g., "au", "ea", "oui").

Assume the program accepts only lowercase values as input.

If no consecutive vowels are present in the string, print "No consecutive vowels present"

Example: "The augmented reality is the near future." → The consecutive groups are: "au", "ea" (in reality), "ea" (in near) = 3

Analysis

What This Question Tests

Pure string traversal. No OOP, no collections. The trick is understanding what "consecutive" means — it's about groups, not individual vowels.

What "Consecutive Vowels" Means
  • NOT counting every individual vowel in the string
  • NOT counting every pair of vowels
  • Counting groups where 2 or more vowels appear next to each other
  • "au" = 1 group. "eau" = still 1 group. "a" alone = 0 groups (needs 2+).
Walkthrough

Character-by-Character Example

Let's trace through: "the augmented reality is the near future"

t  h  e  (space)  a  u  g  m  e  n  t  e  d
                  ^^^^ GROUP 1         (single e's don't count)

r  e  a  l  i  t  y  (space)  i  s
   ^^^^ GROUP 2              (single i doesn't count)

t  h  e  (space)  n  e  a  r
                        ^^^^ GROUP 3

f  u  t  u  r  e
   (single vowels scattered — no groups)
Total consecutive vowel groups: 3
The Classic Mistake

Counting every vowel in the string gives you 12. Counting every pair gives you a different number. The correct answer is 3 groups. A group is a continuous run of 2+ vowels. Once the run ends (a consonant or space appears), you count it as one group and move on.

Algorithm

The Algorithm Step by Step

  • Step 1: Start at index 0. Set count = 0.
  • Step 2: If current character is a vowel, start a sub-loop: keep moving forward while characters are vowels.
  • Step 3: When the sub-loop ends, check: did we move 2 or more positions? If yes, that's one group — increment count.
  • Step 4: Continue from where the sub-loop stopped.
  • Step 5: If current character is NOT a vowel, just move to next character.
  • Step 6: After full traversal, if count is 0, print the "No consecutive vowels" message. Otherwise print the count.
Key Line of Code

if (j - i >= 2) — this is the core logic. i is where the vowel run started, j is where it ended. If the difference is 2 or more, that means 2+ vowels were together. Count it as one group.

Full Code

Complete Solution

Short and clean. The entire solution is under 25 lines.

Solution.java — Complete 
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String input = sc.nextLine().trim().toLowerCase();
        String vowels = "aeiou";
        int count = 0;
        int i = 0;

        while (i < input.length()) {
            if (vowels.indexOf(input.charAt(i)) != -1) {
                // Found a vowel — start scanning the run
                int j = i;
                while (j < input.length() && vowels.indexOf(input.charAt(j)) != -1) {
                    j++;
                }
                // j - i = length of the vowel run
                if (j - i >= 2) {
                    count++;
                }
                i = j; // skip past the entire run
            } else {
                i++;
            }
        }

        if (count == 0) {
            System.out.println("No consecutive vowels present");
        } else {
            System.out.println(count);
        }
    }
}
Line-by-line explanation
1
String input = sc.nextLine().trim().toLowerCase();
Read the full line. trim() removes leading/trailing spaces. toLowerCase() ensures all characters are lowercase for consistent comparison.
2
String vowels = "aeiou";
A string of all vowels. We use indexOf() to check if a character is a vowel. If indexOf() returns -1, the character is not in the string (not a vowel).
3
if (vowels.indexOf(input.charAt(i)) != -1)
Check if the current character is a vowel. charAt(i) gets the character at position i. indexOf() returns its position in "aeiou" or -1 if not found.
4
int j = i; while (j < ... && vowels.indexOf(...) != -1) { j++; }
Inner loop: starting from i, keep advancing j while characters are vowels. When j stops, the vowel run is from index i to j-1.
5
if (j - i >= 2) { count++; }
The magic line. If the run length is 2 or more, count it as one consecutive group. Single vowels (j - i == 1) are ignored.
6
i = j;
Jump i to where j stopped. This skips the entire vowel run we just processed, so we don't double-count.

Watch Out

Common Mistakes

Wrong: Counting all vowels

Counting every vowel individually gives 12 for the first test case.

Counting pairs like (a,u), (e,a), (e,a) seems right but you might accidentally count overlapping pairs in longer runs like "eau" as 2 instead of 1.

The fundamental error: thinking "consecutive" means "individual vowels" rather than "groups."

Correct: Counting groups

Find a vowel. Scan forward until you hit a non-vowel. If the run was 2+ characters long, that's ONE group.

if (j - i >= 2) count++;

This is the key insight. "eau" = 1 group, "au" = 1 group, "a" = 0 groups.

Mistake What Goes Wrong Fix
Counting individual vowels Answer is 12 instead of 3 Count groups of 2+ consecutive vowels, not individual ones
Using for loop with i++ After finding a group, you re-scan the same vowels Use while with i = j to skip past the entire group
Forgetting .toLowerCase() Uppercase vowels (A, E, I, O, U) are missed Convert input to lowercase first, or check both cases
Using == to compare characters against a string Type mismatch error Use vowels.indexOf(ch) != -1 or check against individual chars
